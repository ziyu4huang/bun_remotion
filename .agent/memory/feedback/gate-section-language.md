Graphify Quality Gate section in episode PLAN.md must use **zh_TW as default language**. English is only for professional/technical terminology such as: PASS/WARN/FAIL, nodes, edges, Pipeline, Extract, Merge, Check, Arc, cross-episode links, character IDs (linyi, zhaoxiaoqi etc.), scene IDs (ContentScene1 etc.).

**Why:** The gate section is a story quality review intended for the zh_TW-speaking creator. Mixed English/Chinese makes it harder to read.

**How to apply:** When writing or reviewing any gate section in episode PLAN.md, all descriptive text, findings, assessments, and decisions must be in zh_TW. Technical terms keep English. Apply to both subagent output and manual edits.
