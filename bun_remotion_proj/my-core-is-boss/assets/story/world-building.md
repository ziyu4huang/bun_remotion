# World Building — 我的核心是大佬

## 世界觀

**蒼穹大陸**，修行文明鼎盛。數萬年來，修行者通過感悟天道、修煉靈力追求長生。宗門林立，以「天道宗」為首。

**時間線：** 太平紀元三千年，表面和平實則暗流湧動。

## 天道宗

大陸第一宗門，位於天柱山頂。結構：
- **外門弟子** — 剛入門，基礎修煉
- **內門弟子** — 通過考核，核心培養
- **核心弟子** — 精英中的精英
- **長老** — 各峰主管（煉器峰、劍峰、丹峰、器峰等）
- **宗主** — 最高領袖

**蕭長老** — 煉器峰資深長老，掌管宗門日常運作。

## 宗門政治

- 每百年「百宗大比」決定宗門排名
- 天道宗為首，其次是劍宗、丹宗、器宗
- Ch7 血影門向天道宗宣戰，Ch9 魔尊復活威脅整個大陸

## 核心系統功能

林逸穿越時獲得的神秘系統，來源不明。

### 基礎功能 (v1.0)
- **數據化視角** — 看到所有人的等級、血條、屬性
- **任務面板** — 接取/完成/跳過任務
- **技能系統** — 技能點分配，技能樹
- **物品欄** — 儲物空間
- **自動尋路** — 導航系統

### v2.0 更新 (Ch6)
- **社交系統** — 好友列表、聊天頻道、世界頻道
- **成就系統** — 成就解鎖 + 全服公告
- **商城** — 道具兌換

### v3.0 更新 (Ch10)
- **新地圖** — 其他「伺服器」（其他世界）
- **New Game+** — 世界重置但保留記憶

## 視覺風格

- **配色：** 科技感漸層 `#0a0a2e → #0a1a3e → #1a0a2e`（藍紫基調，區別於鍛造風格）
- **UI 風格：** 遊戲 HUD 風格 — 半透明黑底、模糊背景、螢光邊框
- **系統通知：** 帶有 `[系統]` 標籤的彈窗，左側滑入或底部浮現
- **對話框：** 角色名字左側帶色條，底部顯示，typewriter 打字效果
