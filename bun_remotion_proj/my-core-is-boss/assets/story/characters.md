## 角色詳細

### 林逸 (linyi) — 現代玩家穿越

- **性格：** 隨性、吐槽役、對一切遊戲化理解
- **口頭禪：** 「這也有 bug？」「能不能跳過？」「載入中...」「這遊戲的新手教程在哪？」
- **認知盲區：** 完全不知道自己的「玩家黑話」被所有人解讀為「蘊含天地至理的真言」
- **真實狀態：** 系統基礎面板加成使他超越渡劫期，但他以為自己只是個菜鳥
- **成長弧線：** Ch1-7 獨自穿越 → Ch8 遇到陳默（同類）→ Ch9-10 開始主動利用系統

### 趙小七 (zhaoxiaoqi) — 頭號腦補狂魔

- **性格：** 熱血粉絲、過度解讀專家、寫作狂人
- **口頭禪：** 「太深奧了！」「師兄的意思是...！」「我要記下來！」
- **核心行為：** 把林逸的每句廢話自動昇華成「大道真諦」，撰寫《林逸師兄語錄》
- **成長弧線：** Ch1 第一篇語錄 → Ch2 十萬字日記 → Ch5 賽評員 → Ch9 世界拯救記錄者 → Ch10 新世界篇

### 蕭長老 (xiaoelder) — 漸進崩潰的嚴肅長老

- **性格：** 表面嚴肅、內心戲極多、傳統修行者思維
- **口頭禪：** 「這小子...！」「老夫修行三百年...！」「不可能！」
- **核心行為：** 從懷疑 → 震驚 → 崩潰 → 忠實信徒，偷偷記錄林逸的每句話
- **成長弧線：** Ch1 腿軟 → Ch2 燒功法 → Ch4 被迫當治療 → Ch7 崩潰邊緣 → Ch10 主動當 DPS

### 陳默 (chenmo) — 另一個穿越者 (Ch8+)

- **性格：** 比林逸安靜、程式設計師、偶爾吐槽「你這系統也太作弊了」
- **背景：** 沒有系統，只有前世記憶，一直在勉強裝高手
- **與林逸的關係：** 終於找到可以正常說話的人

## 角色關係動態

```
linyi ←──腦補──→ zhaoxiaoqi    (師兄崇拜，林逸不知情)
linyi ←──誤會──→ xiaoelder     (長老以為他是隱世高人)
linyi ←──理解──→ chenmo        (Ch8+，唯一正常交流對象)
zhaoxiaoqi → 傳播誤解 → 全宗門  (語錄擴散效應)
xiaoelder → 研究記錄 → 古籍對照  (越研究越崩潰)
```

## 對話風格指南

- **林逸：** 簡短、玩家術語、不耐煩、吐槽。「這 NPC...」「跳過跳過」「載入中」
- **趙小七：** 感嘆號多、用「師兄」開頭、主動腦補延伸。「師兄的意思是...！」「太深奧了！」
- **蕭長老：** 文言感、自稱「老夫」、內心矛盾。「這...不可能！」「老夫修行三百年...」
- **陳默：** 程式師語氣、邏輯分析、偶爾吐槽。「你這個系統...邏輯有點問題吧」
- **旁白：** 客觀描述、推動劇情、偶爾冷吐槽。「蕭長老的修行觀念，正在以肉眼可見的速度崩塌。」
