## 系統版本演進

### v1.0 (Ch1-5) — 基礎功能
- 數據化視角、任務面板、技能系統、物品欄、自動尋路
- 穩定運行，偶爾有「bug」（其實是正常功能）
- UI 簡潔，左側面板為主

### v2.0 (Ch6) — 大版本更新
- 新增：社交系統、成就系統、商城
- UI 全面改版，新增右側面板
- 首次出現「全服公告」功能（成就解鎖時）
- 林逸不小心點到新按鈕，觸發一堆新功能

### v3.0 (Ch10) — 新世界
- 解鎖：新地圖（其他「伺服器」/世界）、New Game+
- UI 再次改版，更現代化
- 保留所有歷史數據（成就、語錄、評價記錄）

## GameUI 元件對應

每個系統技能對應的 Remotion 視覺元件：

| 技能 | GameUI 元件 | 預設位置 |
|------|------------|---------|
| 載入中 | LoadingText | 底部中央 |
| 任務面板 | QuestPanel | 右上 |
| 系統通知 | SystemNotification | 頂部 |
| 系統訊息 | SystemMessage | 中央/底部 |
| HP/等級 | HpBar + LevelTag | 角色上方 |
| 成就彈窗 | SystemNotification (type=achievement) | 中央 |
| 社交頻道 | SystemMessage | 底部滾動 |
| 排位段位 | LevelTag | 角色頭頂 |

## 預設配置範例

### 任務通知
```
SystemNotification text="新任務：探索天道宗" type="mission"
SystemNotification text="任務：清剿妖獸（普通）— [跳過] 按鈕已解鎖" type="mission"
SystemNotification text="獎勵已領取：靈石 ×100，經驗值 +500" type="success"
```

### 警告通知
```
SystemNotification text="偵測到高等級單位：Lv.85（隱藏等級：???）" type="warning"
```

### 系統訊息
```
SystemMessage text="蕭長老 評價已更新：敬畏" position="center" color="#A78BFA"
```
