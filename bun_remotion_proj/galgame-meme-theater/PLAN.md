### Episode Structure

Each episode has 6 scenes:
1. **TitleScene** — 開場標題
2. **JokeScene1** — 梗1
3. **JokeScene2** — 梗2
4. **JokeScene3** — 梗3
5. **JokeScene4** — 梗4
6. **OutroScene** — 片尾 + 下集預告

## Adding a New Episode

Follow the `/remotion-best-practices` skill → load `rules/episode-creation.md` for the full workflow.

**Workflow summary:**
1. Read context (PLAN.md, `assets/story/` guides, previous narration.ts)
2. Write story draft → present **confirm block in zh_TW** → wait for user approval
3. After approval → scaffold episode (TODO.md → narration.ts → configs → scenes → update PLAN.md/dev.sh/package.json)
4. Generate TTS → verify in Studio → render MP4

**Config updates (3 places):**
- `scripts/dev.sh` — add to `ALL_APPS` + `get_comp_id()` case
- Root `package.json` — add `start:memeN`, `build:memeN`, `generate-tts:memeN`
- This `PLAN.md` — update Episode Guide table + Commands
