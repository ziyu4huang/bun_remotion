# Plot Lines — 美少女梗圖劇場

## Signature Running Gags

| Gag | Pattern | Episodes | Evolution |
|-----|---------|----------|-----------|
| 減肥/最後一杯 | 小雪說「最後一次」，永遠不是 | ep1, ep2, ep3, ep4 | ep1奶茶→ep2再一局→ep3夜市→ep4手搖飲，每次藉口不同但結果一樣 |
| 小月嘴硬 | 自稱冷靜理性，其實一樣荒謬 | ep1, ep2, ep4, ep7 | ep1「隨便考」98分→ep2連敗10場「隊友問題」→ep7做假的自己去跳舞 |
| 小樱社死 | 社交恐懼 + 暴擊金句 | ep1, ep4, ep6, ep7 | ep1認錯人→ep4想回學校→ep6密碼123456789→ep7「把鹽當糖加了」 |
| 已讀不回 | 現代人社交困境 | ep3, ep4, ep6 | ep3「基本社交技能」→ep6「在嗎連發」+貼圖對話 |

## Gag Evolution Rules

1. **升級（Upgrade）**: 同一個梗出現時，情境更荒謬（減肥→再一局→全部）
2. **反轉（Redirect）**: 梗的受害者換人（小雪減肥→小月連敗→小樱密碼）
3. **收束（Callback）**: Outro 回收本集所有梗，用一句話總結每個笑點

## Character Arc Tracking

### 小雪 — 永恆的承諾者
```
ep1: 「再睡五分鐘」「最後一杯」
ep2: 「最後一局就睡」「打折不買會虧」
ep3: 「減肥是明天的事」
ep4: 「只喝水」「不同的最後一杯」
ep5: 「準時下班=做壞事」
ep6: 「他看了我一眼=喜歡我」
ep7: 數位小雪去上課，真的在家追劇
→ 每集都有一個不會實現的承諾
```

### 小月 — 冷靜理性的幻覺
```
ep1: 「隨便考考」98分，「運氣好而已」
ep2: 連敗10場，「都是隊友的錯」
ep4: 「看天氣預報能看十分鐘」
ep5: 「我再補充一點」→30分鐘
ep6: 已讀不回是基本技能
ep7: 做假的自己去跳舞，數位分身替工作
→ 她的「冷靜分析」往往揭示更荒謬的真相
```

### 小樱 — 社死女王
```
ep1: 認錯人→「我想轉學」「把我埋了吧」
ep3: 過年親戚→「明年我會說我在加班」
ep4: 實習→「我想回學校上課」
ep6: 相親對象62歲
ep7: 「密碼是123456789」→「比原來的還糟」
→ 她的社死和暴擊金句常常是同一段
```

| Priority | Thread | Why |
