# World Building — 美少女梗圖劇場

## Setting

現代台灣校園 + 日常生活。三個美少女室友在大學、網路、夜市、職場、戀愛等場景中經歷荒謬日常。沒有超自然元素，所有笑點來自真實生活中的共鳴。

**Tone:** 輕鬆、共鳴、荒謬感。每個梗都讓觀眾直呼「太真實」。

## Visual Style

- **Galgame visual novel** 風格：角色立繪 + 背景圖 + 對話框
- **漫畫效果 (ComicEffects)**：emoji 反應誇張化（汗滴、愛心、怒氣、哭臉）
- **漫畫音效 (MangaSfx)**：星形爆炸框中的擬聲詞
- **TitleCard**：每集開場標題卡，配合主題色
- **TransitionSeries**：場景間的轉場效果

## Scene Format

每集固定 6 場景：
1. **TitleScene** — 開場標題 + 本集預告
2. **JokeScene1~4** — 四個獨立笑話，每個笑話一場景
3. **OutroScene** — 片尾總結 + 下集預告

每個 JokeScene 通常包含：
- 2~3 個角色對話
- 1~2 個反轉（setup → punchline）
- 至少一個角色負責吐槽（通常是小月）
