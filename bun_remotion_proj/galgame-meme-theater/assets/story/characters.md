## Character Details

### 小雪 (Xiaoxue) — 元氣系美少女

- **外型：** 粉紅雙馬尾，元氣笑容
- **性格：** 熱情奔放、衝動、起話題王、搞事情
- **口頭禪：** 「再一局就睡！」「這是最後一杯！」「我這次是認真的！」
- **弱點：** 每次都說「最後一次」但從來不是最後一次
- **角色定位：** 發起者 — 提出餿主意、引發事件、被吐槽的對象
- **笑點模式：** 她的承諾永遠不會實現（減肥、早睡、戒手搖飲）

### 小月 (Xiaoyue) — 高冷系美少女

- **外型：** 黑長直，冷淡表情
- **性格：** 毒舌、學霸、冷靜分析、吐槽王者
- **口頭禪：** 「那是因為...」「妳的邏輯呢？」「我沒有在生氣，那是興奮。」
- **弱點：** 嘴硬但心軟，嘴上不承認但其實也一樣荒謬
- **角色定位：** 吐槽者 — 補刀、戳破幻想、冷靜分析
- **笑點模式：** 她自以為冷靜理性但其實也很荒謬（連敗十場說是隊友問題）

### 小樱 (Xiaoying) — 天然呆美少女

- **外型：** 橘色短髮，社恐表情
- **性格：** 天然呆、社恐常駐、無意識暴擊金句
- **口頭禪：** 「我想轉學...」「我想消失...」「把我埋了吧...」
- **弱點：** 社交恐懼、密碼是 123456789
- **角色定位：** 暴擊者 — 無意中說出最殘酷的實話
- **笑點模式：** 她說的「深奧道理」其實只是生活中的蠢事，但歪打正著

## Relationship Dynamics

```
小雪 ←→ 小月：互相吐槽，小雪提議小月補刀
小月 ←→ 小樱：小月吐槽，小樱暴擊反殺
小雪 ←→ 小樱：小雪帶風，小樱跟著慌
三人一起：互相拖累的鐵三角，永遠一起犯蠢
```

## Dialog Style Guide

| Character | Style | Example |
|-----------|-------|---------|
| 小雪 | 誇張、感嘆號多、自帶能量 | 「從今天開始減肥，不吃了！」 |
| 小月 | 簡短、犀利、反問句 | 「妳昨天也說是最後一杯喔。」 |
| 小樱 | 害羞、小聲、暴擊收尾 | 「我想轉學...我想消失...」 |

## Visual Guide

- 每個角色只有 **1 張圖**（無表情變體），用 `CharacterSprite` + `ComicEffects` 傳達情緒
- 角色永遠面左（出廠設定），右側角色自動水平翻轉
- 情緒效果：`ComicEffects` emoji（汗滴💧、愛心❤️、怒氣💢、哭臉😭、驚嚇😱）
