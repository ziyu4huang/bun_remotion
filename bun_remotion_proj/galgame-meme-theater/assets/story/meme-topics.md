# Meme Topics — 美少女梗圖劇場

## Joke Categories

### 校園生活 (Campus Life)
- 早八遲到、鬧鐘關七次
- 期末考極限操作、開書考陷阱
- 「沒複習隨便考」→ 學霸98分 vs 學渣28分
- 認錯人、走廊社死
- 讀書會變打遊戲 / 網購

### 網路文化 (Internet Culture)
- Steam 特價囤積症（342個遊戲只打7個）
- 「再一局就睡」→ 天亮
- 連敗不認輸，「都是隊友的錯」
- 已讀不回、貼圖對話
- 「在嗎」連發

### 台灣日常 (Taiwan Daily Life)
- 過年親戚靈魂拷問（幾名？有沒有男女朋友？什麼時候結婚？）
- 夜市吃到撐（「剛吃完晚餐」→「這只是開胃菜」）
- 氣象局不可信（降雨10%→暴雨）
- 手搖飲永遠戒不掉
- 捷運社交距離大師（假裝睡覺、看天氣預報）

### 職場求生 (Workplace Survival)
- 面試包裝（「精通Excel」= 只會合併儲存格）
- 老闆畫大餅三層級（公司會更好→你的努力我看見了→明年上市）
- 開會廢話大全（「我再補充一點」→30分鐘）
- 表演性加班 + 職場量子力學
- 準時下班 = 做壞事

### 戀愛煩惱 (Romance Troubles)
- 暗戀內心戲（「他看了我一眼」= 喜歡我）
- 直男迷惑行為（送電競滑鼠、帶去鹽酥雞、看螢幕當看星星）
- LINE 聊天求生（「正在輸入」10分鐘→「哈」）

### AI 時代 (AI Era)
- AI 代寫作業（3行→30頁，參考文獻編17篇）
- AI 回訊息比本人受歡迎
- 深度偽造（做假的自己替自己工作）

## Joke Formula

每個笑話遵循 **三拍結構 (Setup → Build → Punchline)**：

```
Setup:   小雪提出荒謬計畫（「從今天開始減肥」）
Build:   小月或小樱反應（「妳昨天也這麼說」）
Punch:   反轉或升級（「可是打折不買會覺得虧啊」）
```

**變體：**
- **小樱暴擊版：** Setup → 正常反應 → 小樱無意識說出殘酷實話
- **小月嘴硬版：** Setup → 小月冷靜分析 → 分析揭示她自己更荒謬
- **三人連鎖版：** A提議→B補刀→C暴擊→A反殺

## New Episode Guidelines

新集數的主題選擇方向：
- **共鳴度優先：** 選觀眾會直呼「太真實」的主題
- **台灣文化：** 善用台灣特有場景（夜市、手搖飲、捷運、過年）
- **跨年齡層：** 學生 + 上班族都能共鳴
- **4 梗結構：** 每集 4 個獨立笑話，不連續但主題統一
- **避免重複：** 查 plot-lines.md 確認梗沒用過，或升級後再用
