## 角色詳細

### 周墨 (zhoumo) — 天才工程師煉器師

- **性格：** 工程師思維、模組化設計、忘加按鈕的習慣、對法寶有獨特見解
- **口頭禪：** 「模組化設計」「使用者體驗」「忘加按鈕了」「邏輯修正」
- **核心缺陷：** 每次煉製法寶都會忘記加某個關鍵按鈕/功能
- **成長弧線：** ch1 入宗考試 → ch2 成立邏輯修正小組 → ch3 秘境探索 → ch4 宗門大比

**表情系統（multi-pose）：**
| 表情 | 檔案 | 使用時機 |
|------|------|---------|
| default | zhoumo.png | 日常、思考 |
| angry | zhoumo-angry.json | 不滿、被質疑 |
| nervous | zhoumo-nervous.json | 緊張、麻煩了 |
| shocked | zhoumo-shocked.json | 驚訝、出乎意料 |
| smirk | zhoumo-smirk.json | 得意、自以為聰明 |
| chibi | zhoumo-chibi.png | Q 版搞笑時刻 |

### 考官 (examiner) — 嚴格但崩潰的考官

- **性格：** 正經、按規矩辦事、容易被周墨的法寶搞崩潰
- **口頭禪：** 「這不合規定！」「你這法寶...」
- **成長弧線：** ch1 被飛劍搶儲物袋 → 之後看到周墨就頭痛

**表情系統：**
| 表情 | 檔案 | 使用時機 |
|------|------|---------|
| default | examiner.png | 正式、嚴肅 |
| chibi | examiner-chibi.png | 崩潰 Q 版 |

### 長老 (elder) — 開明但有苦說不出

- **性格：** 看到周墨的創新思維大為讚賞，但經常被周墨的「改良」折騰
- **口頭禪：** 「此子有悟性！」「只是...能不能加個按鈕？」
- **成長弧線：** ch1 破格錄取 → ch2 派任務 → ch3 秘境探索

**表情系統：**
| 表情 | 檔案 | 使用時機 |
|------|------|---------|
| default | elder.png | 長者風範 |

### 陸陽 (luyang) — 投降劍法大師

- **性格：** 溫和、怕事、劍法全是投降招式
- **口頭禪：** 「我認輸」「別打了」「投降是最有效的防守」
- **核心技能：** 投降劍法 — 核心招式「我認輸」「別打了」，防守效率極高（因為對手笑到沒力氣）
- **成長弧線：** ch2 加入邏輯修正小組 → ch3 秘境探索

**表情系統（multi-pose）：** 有多個表情變體（.json 檔案）

### 孟景舟 (mengjingzhou) — 單身光環研究者

- **性格：** 學術型、愛寫論文、分析能力強
- **口頭禪：** 「我研究了...」「根據我的論文...」
- **核心技能：** 單身光環（被動技能，方圓三丈內女性自動遠離）
- **成長弧線：** ch2 加入邏輯修正小組 → ch3 秘境探索

**表情系統（multi-pose）：** 有多個表情變體（.json 檔案）

### 雲芝 (yunzhi) — 師姐 (ch4+ planned)

- **性格：** 嚴格、專業、對法寶有標準評估
- **口頭禪：** 「設計理念很實用，但執行方式有問題」
- **登場：** ch4-ep1 飛舟事件

## 角色關係動態

```
zhoumo ←──組隊──→ luyang, mengjingzhou    (邏輯修正小組)
zhoumo ←──指導──→ elder                   (長老賞識 + 無奈)
zhoumo ←──考核──→ examiner                (考官 + 受害者)
zhoumo ←──評估──→ yunzhi                  (ch4 師姐)
```

## 對話風格指南

- **周墨：** 工程師術語、模組化、直接。「使用者體驗」「模組化設計」「邏輯修正」
- **考官：** 官方語氣、崩潰。「不合規定」「你這法寶...」
- **長老：** 長者語氣、文言感。「此子有悟性」「三百年來...」
- **陸陽：** 溫和、投降。「我認輸」「別打了」
- **孟景舟：** 學術、論文。「根據我的研究」「論文第 X 篇」
- **旁白：** 客觀描述、偶爾冷吐槽。
